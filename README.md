# [Stated Plainly](https://substack.com/@statedplainly/posts) :: [Odin](https://odin-lang.org/)
Stated plainly, this is a dumping ground for everything I write in [Odin](https://odin-lang.org/).

## Sections
- [apps](https://github.com/stated-plainly/sp__odin/tree/main/apps) - Applications that can be run via `odin run .`
- [libs](https://github.com/stated-plainly/sp__odin/tree/main/libs) - Libraries that can be imported via `import {optional alias} "{path to package}"`
- [reference](https://github.com/stated-plainly/sp__odin/tree/main/reference) - Reference docs explaining aspects of [Odin](https://odin-lang.org/)
- [tests](https://github.com/stated-plainly/sp__odin/tree/main/tests) - Library crate tests that can be run via `odin test . -all-packages`
